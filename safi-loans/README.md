# SafiLoans Frontend

React + Vite frontend for the SafiLoans system.

## Run locally

1. Install dependencies:
   - `npm install`
2. Copy `.env.example` to `.env` and set:
   - `VITE_API_BASE_URL=http://127.0.0.1:8000/api`
3. Start development server:
   - `npm run dev`

## Build

- `npm run build`

## Main routes

- `/` Home
- `/apply` Loan application
- `/mgmt-panel` Management dashboard
